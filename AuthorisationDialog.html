Using this 'Add-On' means connecting it to Github via OAuth. This happens the first time you click on 'Start Publishing', when you grant permissions for Gabriel to publish to your repositories on your behalf. Currently, this means you can only be 'signed' into a single account at any one time. If you would like to publish using a different account, you will need to click the button below to begin the authorisation process again.
: You have already authorised this add-on to access your GitHub Account. You can view the details of this application on your GitHub account. If you would like to change the account you are using to access GitHub, please sign out of GitHub and click on the 'Authorise' button below.
